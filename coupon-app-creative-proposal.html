AI 拍照识别

拍照即识别，自动提取优惠券名称、平台、金额和有效期。内置 OCR 引擎支持中英文，无需上传云端，保护隐私。

本地处理 · 隐私安全

智能过期提醒

支持钉钉、飞书双渠道提醒。自定义提前提醒天数，在过期前自动通知，不遗漏任何一张券。

多平台推送 · 可自定义

自动云端同步

数据自动同步到你的 GitHub Gist，云端备份防丢失。更换设备也能无缝继续使用。

GitHub Gist · 零成本

每日自动检查

基于 GitHub Actions 实现每日自动检查。即使不打开网站，到期前也会收到提醒。

效率提升
