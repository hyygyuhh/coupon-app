羊毛管家

优惠券智能管理助手 - 让每一分优惠都不被错过

📷

AI 拍照识别

采用 Tesseract.js 引擎，智能识别优惠券信息，自动提取名称、平台、金额和有效期，省去手动输入的繁琐。

🔔

过期提醒

支持钉钉、飞书双渠道提醒，自定义提前提醒天数，确保你在优惠券过期前及时使用，不浪费任何权益。

🔄

自动同步

本地数据自动同步到 GitHub Gist，云端备份防丢失，随时随地访问你的优惠券收藏。

☁️

自动提醒

基于 GitHub Actions 实现每日自动检查，即使不打开网站也能收到过期提醒。
